        Cloud sync

        Syncs across all your devices using Firebase. Free. No login needed.

        ○ Not connected

        Same passphrase on every device = same data.

        How to set up

        1. Go to console.firebase.google.com → Create a project
        2. Click the Web icon (</>) → Register app → copy the firebaseConfig object
        3. In Firebase: Build → Realtime Database → Create database → test mode
        4. Paste the config above + set a passphrase → Save & connect
        5. Repeat on every device using the same passphrase

        Install as an app

        MacBook (Chrome/Edge): Address bar → install icon → Install.

        iPhone (Safari): Share → Add to Home Screen.

        Import iCloud calendar

        1. Mac: open Calendar app → click a calendar in sidebar
        2. File → Export → Export... (saves a .ics file)
        3. Dashboard: Calendar tab → Import .ics → pick the file
